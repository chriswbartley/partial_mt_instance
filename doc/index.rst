Welcome to partial_mt_instance's documentation!
====================================

`PartialInstanceClassifier` is a Random Forest classifier with the added capability of partially monotone features. It is very fast and moderately accurate. 

To install, simply use :bash:`pip install partial_mt_instance`. For full documentation you've come to the right place. For a brief overview, refer to the `README file 
<https://github.com/chriswbartley/partial_mt_instance/blob/master/README.md>`_ in the Github repository.
